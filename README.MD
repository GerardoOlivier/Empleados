## Descripción

Sistema de Empleados es una aplicación web desarrollada con Spring Boot que permite gestionar un información de empleados mediante operaciones CRUD 
(Listar, Agregar, Editar y Eliminar). La aplicación utiliza Spring MVC con JSP/JSTL para la capa de presentación,
Bootstrap para el diseño responsive y Spring Data JPA para la persistencia en PostgreSQL.

## Características Principales

- Operaciones CRUD completas para empleados
- Interfaz de usuario con JSP, JSTL y Bootstrap 
- Diseño responsive y moderno
- Persistencia con Spring Data JPA y PostgreSQL
- Validación de formularios
- Arquitectura MVC (Modelo-Vista-Controlador)
